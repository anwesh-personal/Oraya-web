# ORAYA LOGOS

This folder contains official Oraya logo files in various formats and variants.

## Logo Variants

- **Full Logo** - "Oraya" wordmark with icon
- **Icon Only** - Symbol for favicons and app icons  
- **Monochrome** - Black/white versions for print
- **Gradient** - Primary to accent gradient version

## File Formats

- **SVG** - Vector format (preferred for web/print)
- **PNG** - Raster format with transparency (multiple sizes)
